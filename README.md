# Cute Docu Shelf

## Project Overview
A Streamlit-based document management system that acts as a bridge between Excel files and a Postgres database.

## Tech Stack
- **Language**: Python
- **Database**: Postgres
- **UI Framework**: Streamlit
- **Data Source**: Excel
- **Version Control**: Git

## Key Features
1. **Import**: Load data from Excel files into Postgres.
2. **Export**: Dump data from Postgres back to Excel.
3. **Manage**: A multi-page web interface to view and manage documents.
